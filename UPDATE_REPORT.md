## 🚀 使用方法

### 基础用法

```bash
cd C:\Users\Administrator\.openclaw\workspace\skills\video_smart_editor
uv run python video_editor.py
```

### 自定义配置

编辑 `video_editor.py` 顶部：

```python
INPUT_DIR = Path(r"E:\video_input")
VIDEO_NAME = "你的视频名"
WHISPER_MODEL = "base"  # tiny/base/small/medium/large
```
